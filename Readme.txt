【关于不正常工作的说明】请先手动去pt.3g.qq.com手动登陆QQ账号聊天，如能正常使用，则程序也会正常工作。
如不能正常使用，说明你的QQ账号状态异常，多登陆几次就好了。
/s?q=anyway可以模拟重复登录这一过程，当然你手动登陆也行。
